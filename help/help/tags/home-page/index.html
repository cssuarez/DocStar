Navigation Bar

By dibuono · May 9, 2012 · 0 Comments

The Navigation Bar at the top of the Eclipse3 window features tabs for the five main sections of the application.

The Navigation Bar is always visible at the top of the window

Click on a tab to access that section.

  • Home
  • Retrieve
  • Capture
  • Workflow
  • Admin

NOTE:Â  The tab for the section you are currently viewing displays in yellow.
