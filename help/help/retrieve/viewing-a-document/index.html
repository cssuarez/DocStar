Viewing a Document

  • Page
  • Discussion
  • History
Help Page > Retrieve > Viewing a Document
  • MetaData:
    • Able to edit the currently viewed documents metadata.
    • Able to change the Documents:
      • Content Type.
      • Title.
      • Keywords.
      • Security Class.
      • Inbox (Able to add to 0 or more inboxes)
      • Folder(Able to add to 0 or more folders)
      • Fields:
        • Fields generated by user.
  • Change Security of selected folder, inbox, or document.
    • Only able to change the security of one item at a time.
  1. View
    • This will open up the document(s) in the “View” pane. Click here for details on the “View” pane.
  2. Save a Copy
    • This will download the selected file(s) in a .zip file called “Documents”.
    • Depending on your browser and system settings, it will either download to your default location or allow you to specify where to download the file.
    • Depending on your browser and system settings, subsequent downloads will be called “Documents(1).zip, Documents(2).zip, etc” or ask you to rename the file.
  3. Merge
    • You must select two or more documents to merge.
    • This will merge the selected documents into one single document, retaining the name of the first document in the list.
    • The other documents will be deleted after the merge.
  4. E-mail Attachment
    • This will e-mail the selected document(s) to addressees of your choice.
    • The document(s) will be added to a .zip file called “Documents.zip”.
    • Type the subject of your email in the “Subject” field.
    • Type the e-mail addresses of your intended recipients in this field; separate multiple addresses with a semicolon.
    • Click the “Send” button; an “E-mail Sent” message will appear briefly.
    • Click the “Close” button to close this dialog box.
    • The sender of this email will appear as “support” in your recipient(s) inbox.
  5. E-mail Attachment w/Password
    • This will allow you to first create a password for the document(s) to be sent.
    • Type a password and click “Next”.
    • The document(s) will be added to a .zip file called “Documents.zip”.Type the subject of your email in the “Subject” field.
    • Type the e-mail addresses of your intended recipients in this field; separate multiple addresses with a semicolon.
    • Click the “Send” button; an “E-mail Sent” message will appear briefly.
    • Click the “Close” button to close this dialog box.
    • The sender of this email will appear as “support” in your recipient(s) inbox.
  6. E-mail Link
    • This will send a link to the selected document(s) to the recipient(s) of your choice.
    • Type the subject of your email in the “Subject” field.
    • Type the e-mail addresses of your intended recipients in this field; separate multiple addresses with a semicolon.
    • Click the “Send” button; an “E-mail Sent” message will appear briefly.
    • Click the “Close” button to close this dialog box.
    • The sender of this email will appear as “support” in your recipient(s) inbox.
    • The link will be a downloadable “documents.zip” file.
  7. E-mail Link w/Password
    • This will allow you to first create a password for the document link(s) to be sent.
    • Type a password and click “Next”.
    • Type the e-mail addresses of your intended recipients in this field; separate multiple addresses with a semicolon.
    • Click the “Send” button; an “E-mail Sent” message will appear briefly.
    • Click the “Close” button to close this dialog box.
    • The sender of this email will appear as “support” in your recipient(s) inbox.
    • The link will be a downloadable “documents.zip” file.
  8. Remove
    • Deletes the selected folder or inbox, but the contained documents are not deleted.
    • Contained documents are still searchable. EXPLAIN
  9. Delete
    • Deletes the selected folder or inbox as well as any contained documents.
    • Contained documents are no longer searchable. EXPLAIN
 
  • E-mail Attachment:
    • Create an e-mail with a subject to be sent to a specified e-mail address.
    • The e-mail will contain an attachment of the selected document(s).
  • E-mail Attachment with password:
    • Create an e-mail with a subject to be sent to a specified e-mail address.
    • The e-mail will contain an attachment of the selected document(s).
    • The zip file that is attached isn’t password protected.
    • It is able to be opened without supplying a password.
    • The supplied password gets applied to each of the selected documents within the attached zip file.
  • E-mail Link:
    • Create an e-mail that contains a link to download the selected files.
  • E-mail Link with password:
    • Create an e-mail that contains a link to download the selected files.
    • Upon navigation to the link the navigator is prompted for the supplied password in order to download the selected files.
  • Client Service is not installed/Running:
    • User is prompted to download the selected documents.
    • The user can then print the documents as normal.
  • Client Service is Installed/Running:
    • User is prompted with a print dialog.
  • Viewer Context menu:
    • When being viewed in our viewer, not natively, a user is able to right click the image of the document and bring up a context menu.
    • Recognition:
      • Barcode:
        • User is able to draw a rectangle to detect a barcode.
    • Zoom:
      • Able to zoom in or out on the currently viewed document.
      • Scrolling mouse wheel (up: zoom in, down: zoom out) will also zoom in our out at the same percentages.
