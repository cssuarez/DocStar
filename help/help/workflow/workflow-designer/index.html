Workflow Designer

  • Page
  • History
Help Page > Workflow > Workflow Designer
  • The workflow designer allows the user to:
    • Create a new workflow by selecting –New–  from the presented dropdown.
    • Edit the currently selected workflows information including:
      • Name
      • Description
      • Exception Assignee
      • Security Class
    • Save the workflow
      • Saves the currently selected workflow
      • If new the user is prompted to give the workflow a unique title
    • Save As…
      • Saves the currently selected workflow with a specified title
    • Delete
      • Delete the currently selected workflow
  • Editing the workflow:
    • Start:
      • A starting step to the workflow
      • Hovering over will present the user with the following options:
        • Edit the current workflow information (same as clicking edit button)
        • Add new step
    • End:
      • An end point representation
    • Hovering over steps will present the user the options to
      • Edit the step being hovered over
      • Delete the step being hovered over
      • Or add an additional step, that will be the destination step for the step being hovered over
  • Editing Steps:
    • Change the steps name
    • Change the assignee mode
      • Owner, current user using the workflow
      • User, select from a list of users
      • Round Robin, select from a list of groups
      • Random, select from a list of groups
      • Load Balance, select from a list of groups
      • Whole Group, select from a list of groups
    • Set who the assignee should be based upon the chosen assignee mode
    • Determine the branch mode
      • Automatically determines next branch
      • User selects next branch
    • Edit Actions:
      • Add, Edit, or Delete an action
        • Add an action
          • click the add action button
          • user is prompted with an action editor (same as editing an existing action)
        • Editing an action
          • The user will be prompted with an Action Editor
            • User is able to:
              • Change the actions name
              • Add, Edit, or Delete a task
                • Add
                  • Drag and drop a task from the task list on the left side of the editor into the tasks list
                • Edit
                  • Edit the currently selected tasks values
                • Delete
                  • Delete the currently selected task
              • Adding and Editing a task will display a task editor to the user.
              • Add the current action to an action library
        • Deleting an action
          • Deletes the currently selected action
    • Edit Branches:
      • Add, Edit, or Delete a branch
      • Add Branch
      • Edit Branch
        • Editing of a branch is done within the step editor
        • Change the label
        • Change the condition
        • Change the description
        • Change which step should be the next step for the selected branch
        • Change what line type the branch should have (straight, flow chart, Bezier)
      • Delete Branch
        • Delete the currently selected branch
  • Edit Tasks:
    • Able to change the current task by selecting from the task dropdown
    • Input and Output Arguments
      • Able to begin typing in and get auto complete  for built in arguments
      • Able to drag and drop an argument from the arguments library on the left hand side of the editor
      • User is able to type in whatever they want
        • It will be added to the arguments library when the task is saved.
